# Advanced Table
- 테이블 작성을 좀 더 쉽게 만들어줌


---
# 사용법

- 아래와 같은 형식으로 헤더 입력 후 엔터를 치면 바디가 자동을 생성
```
| 제목1 | 제목2 | 제목3 |
```

- 컨텐츠의 길이에 따라 테이블 셀의 크기가 변하게 되는데 레이아웃을 자동으로 업데이트함


---
# 단축키
- tab : 다음셀
- shift + tab : 이전 셀
- 엔터 : 다음 줄
- ctrl + shift + d : 테이블 컨트롤 패널 열기



---
# 컨트롤 패널
-  셀의 좌, 우, 센터 정렬
- 행, 열 위치 변경
- 셀 내용 정렬
- 등등
